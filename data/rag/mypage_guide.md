# 마이페이지 사용 가이드

## 확인 가능한 항목
마이페이지에서는 관심 행사, 참여 행사, 문의 내역 등을 확인할 수 있다. 로그인한 사용자에게만 개인화 데이터가 제공된다.

## 챗봇 연동 범위
챗봇은 마이페이지 관련 질문을 받으면 다음 순서로 답한다.
1. 기능 위치와 사용 방법 설명
2. 로그인 여부 확인
3. 연결된 API가 있으면 최근 상태 요약

## 내 상태 확인
내 상태 확인은 단순한 설명이 아니라 실제 개인 데이터 조회에 가까운 질문이다. 챗봇은 참여 행사 수, 관심 행사 수, 문의 수를 요약할 수 있고, 정확한 결제 처리 단계는 아직 별도 API 연결 여부에 따라 달라질 수 있다고 안내한다.
